Decky AI exposes 7 tools via MCP. Your AI assistant calls these automatically when you ask it to generate or manage decks.

## Typical workflow

```mermaid
flowchart TD
    A[list_masters<br/>optional: pick a template] --> B[create_deck<br/>start job, reserve credits]
    B --> C[get_deck_status<br/>poll every ~10s]
    C -->|succeeded| D[get_deck_download_url<br/>get .pptx link]
    C -->|still running| C
```
